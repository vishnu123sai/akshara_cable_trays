- **Horizontal runs:** Maximum 1.5m - 2.4m (varies by width)

## LOAD CALCULATION EXAMPLES

### Example 1: Ladder Tray Load Assessment
- Tray width: 400mm (Standard)
- Max load capacity: 350kg/m
- Installation length: 6m
- Total cable weight: 800kg

**Support requirement:**
- Minimum supports: 800kg ÷ 350kg = 2.3 → 3 supports minimum
- Spacing: 6m ÷ 3 = 2m (exceeds max 1.8m)
- **Solution:** Use 4 supports at 1.5m intervals

### Example 2: Support Bracket Selection
- Tray width: 600mm
- Estimated cable load: 400kg
- Mount location: Wall
- Required spacing: 2.0m

**Bracket selection:**
- Standard bracket: 500kg capacity ✓ Suitable
- Mounting type: Cantilever or Channel
- Fastening: M16 bolts into concrete
